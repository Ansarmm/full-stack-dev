Облачные провайдеры: DigitalOcean, PS Cloud (ps.kz) и Google Cloud

DigitalOcean
DigitalOcean – популярный международный облачный провайдер, ориентированный на разработчиков. Его виртуальные машины называются Droplets.

DigitalOcean предлагает несколько типов Droplets: 

- Стандартные (Basic) для простых приложений
- Сбалансированные General Purpose
- Оптимизированные по памяти (Memory-Optimized)
- По ЦП (CPU-Optimized)
- По диску (Storage-Optimized) 
- По GPU для вычислений

В качестве операционных систем доступны все распространённые Linux-дистрибутивы (Ubuntu, Debian, CentOS, Fedora, Rocky, AlmaLinux и др.)

Масштабирование возможно как вертикально (resize Droplet для увеличения CPU/ОЗУ), так и горизонтально – через развёртывание нескольких Droplets за Load Balancer и автоскейлинг пулов



Google Cloud Platform (GCP)
Google Cloud — крупный облачный провайдер от Google с гибкими и масштабируемыми решениями для любых проектов.
Виртуальные машины называются Compute Engine Instances.

Типы VM в GCP:

E2, N2, N2D – стандартные для общего назначения
C2, C3 – высокопроизводительные, оптимизированные по CPU
M1, M2, M3 – оптимизированные по памяти
A2, T4, L4 и др. – с GPU для вычислений
T2D – энергоэффективные бюджетные VMs (на базе AMD)

Операционные системы: Ubuntu, Debian, CentOS, Rocky Linux, SUSE, Windows и др.

Масштабирование: вертикальное (resize VM) и горизонтальное (автоскейлинг групп экземпляров), продвинутые балансировщики нагрузки.



PS Cloud (ps.kz)
PS.kz — локальный облачный провайдер из Казахстана с простой панелью и возможностью точной настройки серверов.
Виртуальные машины просто называются Облачные серверы (Cloud VM).

Типы серверов:
Пользовательская настройка CPU, RAM и диска вручную
Отдельные варианты с GPU и быстрым NVMe-хранилищем

Операционные системы: Ubuntu, CentOS, Debian, AlmaLinux, Windows Server

Масштабирование: вручную — можно изменить параметры сервера или создать дополнительные экземпляры, но нет автоматического автоскейлинга.



Выбранный провайдер: DigitalOcean

Выбранный сервер:
2 vCPU, 4 GB RAM, 80 GB SSD
Тип: Basic Droplet

Стоимость: $24/месяц

Причина выбора:
DigitalOcean предлагает простую и стабильную инфраструктуру с понятной ценовой моделью. Он идеально подойдёт для небольшого веб-приложения или API, позволяя масштабироваться по мере роста без сложной настройки.
