1. Fortran (1957) Интересный факт: 
    Один из первых языков, предназначенных для научных вычислений. Его название расшифровывается как "Formula Translation".
2. Lisp (1958)
    Интересный факт: Lisp считается одним из самых старых языков программирования и является основой для многих современных языков, таких как Python.
3. COBOL (1959)
    Интересный факт: COBOL был разработан для бизнес-приложений и используется в банковской сфере и правительственных системах до сих пор.
4. Pascal (1970)
    Интересный факт: Назван в честь математика Блеза Паскаля. Он был создан для обучения программированию и активно использовался в университетах.
5. C (1972)
    Интересный факт: Язык C стал основой для многих других языков, включая C++, C# и Java. Он используется для системного программирования.
6. Python (1991)
    Интересный факт: Python был создан для повышения читаемости кода, и его синтаксис вдохновлен языками ABC и Modula-3.
7, Java (1995)
    Интересный факт: "Напиши один раз, запускай везде" — слоган Java, благодаря чему он стал популярным для веб-разработки и мобильных приложений.
8, JavaScript (1995)
    Интересный факт: JavaScript изначально назывался Mocha и стал основным языком для создания интерактивных элементов на веб-страницах.
