# Детали проекта to-do list manager

## Сущности
1. Task 
2. TaskList
3. SubTask

## Операции с сущностями
### TaskList
* Создавать
* Просматривать
* Удалять
* Обновлять

### Task
* Создавать
* Пометить выполненным
* Удалять
* Добавить в TaskList

### SubTask
* Создавать
* Пометить выполненным
* Удалять
* Добавить в Task

## Пользователи
Пользователи могут управлять всеми сущностями и проводить все возможные операции
